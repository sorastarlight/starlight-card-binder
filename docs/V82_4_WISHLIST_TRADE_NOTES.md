# V82.4 — Wishlist & Trade Lists

Adds a secure collector-facing wishlist and duplicate-only “For Trade” list inside the unified Binder shell.

## Install
1. Run `docs/supabase/v82_4_wishlist_trade_lists.sql` in Supabase SQL Editor.
2. Replace the matching `docs` files in the repository.
3. Commit and push.

The first owned copy of every card is protected. Only quantities above one can be listed for trade.
