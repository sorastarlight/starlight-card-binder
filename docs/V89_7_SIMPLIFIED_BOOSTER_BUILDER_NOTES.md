# V89.7 — Simplified Booster Builder

## Install

1. Run `docs/supabase/v89_7_simplified_booster_builder.sql` in Supabase SQL Editor.
2. Replace the deployed `docs` files with this package.
3. Hard-refresh the site.

## Highlights

- Start new boosters from friendly templates.
- Copy an existing booster, including slots, odds, reward cards, artwork, and pricing.
- Copied and template boosters begin inactive.
- Plain-language pack preview.
- Visual rarity totals with 100% validation.
- Automatic balancing and recommended rarity presets.
- Card pull weights are now shown as friendly Pull Frequency choices.
- Advanced numeric weights remain available when needed.
