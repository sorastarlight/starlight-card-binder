# V90.0 — Starlight Card Database 2.0

## Highlights
- Pokémon-style card classification: Category, Subcategory, Rarity, Variant, Finish, Distribution, Tags, Collector Number, and Publish Status.
- Existing cards are safely migrated to Character / Standard Art / Standard Finish defaults.
- Guided card editor and searchable database filters.
- Booster Builder 2.0 with Standard, Generous, and Premium default pull-rate presets.
- Optional series, category, finish, promo, and duplicate filters for booster pools.
- Advanced per-slot percentages remain available behind Advanced Mode.
- Public catalog includes the new classification fields.

## Install
Run `supabase/v90_0_starlight_card_database_2.sql`, then deploy the included site files.
