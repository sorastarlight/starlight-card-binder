# V90.2.1 — Subcategories, Franchises & Themes

1. Run `supabase/v90_2_1_subcategories_franchises.sql` in Supabase SQL Editor.
2. Replace the matching deployed files.
3. Hard-refresh with `Ctrl + F5`.

The new **Categories & Subcategories** tab is inside **Starlight Card Management**.

Subcategories are independent from Categories. A card can therefore use:

- Category: `Outfit`
- Subcategory: `Sonic The Hedgehog`

The "Suggested Category" on a subcategory is only used to organize the list.
