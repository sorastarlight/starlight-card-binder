# V79.4 Polish Notes

## Holographic effect
- Kept the animated holographic system, but softened the bright white overlay so the card illustration remains easier to see.
- Added a new pulsing rainbow shine layer with lower opacity and softer blending.
- Reduced star sparkle brightness so the stars feel magical without washing out the art.

## Booster splash page
- Increased the starting booster pack art size by roughly 10% from the V79.3 baseline.
- Slightly widened the adaptive pack grid so future booster packs still scale cleanly.

## Booster select button
- Slimmed down the pink button under each booster image.
- Reduced padding, border thickness, line spacing, and text size.
- Kept `seriesId` and `seriesName` together on one line, with collection count underneath.
