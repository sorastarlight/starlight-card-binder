# V82.8.5 Firefox Performance Mode

- Automatically detects Firefox and enables a lighter visual mode.
- Removes expensive backdrop blur and continuous sheen effects in Firefox.
- Pauses animations when the tab or content is offscreen.
- Adds lazy loading and async decoding to non-critical images.
- Reduces favorites sync polling from 500ms to 1500ms and pauses it while hidden.
- Adds layout/paint containment and content visibility to long card grids.
- Keeps reward reveals and core magical presentation intact.
